指针与引用

解引用,给*p赋值实际上a=12

引用传递，调用函数时，对引用内的所有操作会直接修改原始实参，而非像值传递那样拷贝实参副本

Const

const是只读限定符，修饰普通变量时，变量无法重新赋值，const修饰指针，需根据*位置判断.

Static

静态成员变量，函数，全局统一

面向对象

构造函数初始化

可以调用析构函数销毁

梦到对象就活过来，醒来对象就消失，析构执行流眼泪。

  • c 不是独立变量，而是 a 的 “别名”，它与 a 指向同一块内存空间，不占用额外内存（逻辑上）。b 是一个独立的 int 变量，它会在内存中开辟一块新的空间，存储的是 a 的值（即 5）
  • c是p写法的语法糖
  • const引用节省存储空间

面向对象之运算符重载

static_cast(a)是 C++ 中的静态类型转换运算符，用于在编译阶段完成类型转换，是最常用的类型转换方式之一。
